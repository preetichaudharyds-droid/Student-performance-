# 📊 Student Performance Analysis

## 📌 Overview

This project focuses on analyzing student performance data to identify key factors that influence academic success. Using data analysis and machine learning techniques, the goal is to uncover patterns and build predictive models for student outcomes.

---

## 🎯 Objectives

* Analyze student academic performance
* Identify factors affecting scores (study time, parental education, etc.)
* Perform data cleaning and preprocessing
* Build predictive models to estimate student performance
* Visualize insights for better understanding

---

## 🗂️ Dataset

The dataset contains information such as:

* Gender
* Study Time
* Parental Education
* Lunch Type
* Test Preparation Course
* Math, Reading, and Writing Scores

---

## 🛠️ Technologies Used

* **Python**
* **Pandas** – Data manipulation
* **NumPy** – Numerical operations
* **Matplotlib / Seaborn** – Data visualization
* **Scikit-learn** – Machine learning models

---

## 🔍 Project Workflow

1. **Data Collection**
2. **Data Cleaning**

   * Handling missing values
   * Encoding categorical variables
3. **Exploratory Data Analysis (EDA)**

   * Distribution of scores
   * Correlation analysis
4. **Feature Engineering**
5. **Model Building**

   * Linear Regression
   * Decision Tree
   * Random Forest
6. **Model Evaluation**

   * Accuracy
   * Mean Absolute Error (MAE)
   * R² Score

---

## 📈 Key Insights

* Students who completed test preparation performed better
* Parental education has a positive impact on scores
* Study time strongly correlates with performance
* Lunch type also shows influence on results

---



---

## 📊 Results

* Achieved good prediction accuracy using Random Forest
* Identified key factors impacting student performance
* Built a reliable model for score prediction
